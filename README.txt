Compiling of the code:

Open "converter.py" file in this folder
Paste the smiles formula of the code in the given space
Before running the code, refer to the packages section of the REPORT
Compile and run the code using command "./run.sh"

To convert from ball-stick diagram to space filling diagram:
Run the code
Right click on the window that pops up
Click on space filling diagram
